About Me

I earned a Bachelor's degree in Accounting and Economic Informatics (IT) in which I learned about databases such as Access, Lotus 1,2,3, and FoxPro. In my first job out of college, I designed and developed a database software application in the FoxPro language for the company. The software reduced more than 60% of processing time for accounting documents and it was the only package of software used at the company for eight years.


After discovering a passion for programming, I decided to attend a coding bootcamp to gain skills in the C#/.NET stack. I researched and ultimately chose to attend Tech Elevator Code School because of the quality of instruction. There I acquired new software language programming skills: C#, .NET, MS SQL Server, ASP.NET MVC5, HTML5 CSS3, Java Script and JQuery and used them to build a full stack web applications.
